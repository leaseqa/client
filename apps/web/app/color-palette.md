# Color Palette - Minimal & Fresh Design

## 60-30-10 Color Rule

### 60% - Dominant Color (Background)
- **Primary Background**: `#f8fafc` - Soft blue-gray (main page background)
- **Secondary Background**: `#f1f5f9` - Slightly darker for subtle sections
- **Usage**: Page backgrounds, large empty spaces

### 30% - Secondary Color (Content Areas)
- **Card Background**: `#ffffff` - Pure white
- **Panel Background**: `#f9fafb` - Very light gray for subtle panels
- **Usage**: Cards, content containers, panels

### 10% - Accent Color (Highlights & CTAs)
- **Primary Accent**: `#6366f1` - Soft indigo (main CTA buttons, links)
- **Secondary Accent**: `#8b5cf6` - Soft purple (secondary actions)
- **Success**: `#10b981` - Fresh green
- **Warning**: `#f59e0b` - Warm amber
- **Error**: `#ef4444` - Soft red
- **Usage**: Buttons, links, icons, highlights

## Neutral Colors

### Text Colors
- **Primary Text**: `#1e293b` - Dark slate (headings, main text)
- **Secondary Text**: `#64748b` - Medium slate (body text)
- **Muted Text**: `#94a3b8` - Light slate (hints, labels)
- **Inverse Text**: `#ffffff` - White (on dark backgrounds)

### Border Colors
- **Primary Border**: `#e2e8f0` - Light gray-blue
- **Secondary Border**: `#cbd5e1` - Medium gray-blue
- **Subtle Border**: `#f1f5f9` - Very light gray-blue

## Gradient Options (Soft & Subtle)

### Primary Gradient
- `linear-gradient(135deg, #6366f1 0%, #8b5cf6 100%)` - Indigo to purple

### Success Gradient
- `linear-gradient(135deg, #10b981 0%, #059669 100%)` - Green gradient

### Info Gradient
- `linear-gradient(135deg, #3b82f6 0%, #2563eb 100%)` - Blue gradient

### Warning Gradient
- `linear-gradient(135deg, #f59e0b 0%, #d97706 100%)` - Amber gradient

## Color Application Examples

### Backgrounds
- Page background: `#f8fafc` (60%)
- Card background: `#ffffff` (30%)
- Hover states: `#f1f5f9`

### Buttons
- Primary button: `#6366f1` (10% accent)
- Secondary button: White with `#e2e8f0` border
- Text button: Transparent with `#6366f1` text

### Icons & Highlights
- Primary icon: `#6366f1`
- Secondary icon: `#64748b`
- Success icon: `#10b981`
- Warning icon: `#f59e0b`
- Error icon: `#ef4444`
